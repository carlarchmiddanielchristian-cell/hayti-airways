# Hayti Airways

Site web d'une compagnie aérienne haïtienne fictive, réalisé dans le cadre du cours de **Programmation d'applications web 1** à l'Université Quisqueya (Session 2025-2026).

---

## Aperçu

**Hayti Airways** est une compagnie aérienne nationale haïtienne fondée en 1994, connectant Haïti à plus de 35 destinations dans les Caraïbes, l'Amérique du Nord et l'Europe.

## Pages du site

| Page | Description |
|------|-------------|
| `index.html` | Accueil avec hero, carrousel, services et témoignages |
| `apropos.html` | Histoire et valeurs de la compagnie |
| `services.html` | Détail des services offerts |
| `projets.html` | Liste des destinations |
| `contact.html` | Formulaire de réservation/contact |

## Améliorations JavaScript (Examen Final)

Le fichier `js/main.js` ajoute les fonctionnalités suivantes :

- **Barre de progression** — barre dorée en haut qui avance au scroll
- **Bouton retour en haut** — apparaît après 300px de défilement
- **Compteurs animés** — les statistiques (2M+, 98%…) s'animent à l'écran
- **Validation du formulaire** — vérification en temps réel des champs de contact
- **Toast de notification** — confirmation visuelle après soumission
- **Filtre des destinations** — filtrer par région (Caraïbes, Amérique, Europe)
- **Animation au scroll** — les cartes apparaissent en glissant vers le haut
- **Recherche avec autocomplete** — barre de recherche dans le hero

## Technologies utilisées

- HTML5, CSS3
- JavaScript (Vanilla JS — sans framework)
- Bootstrap 5.3
- Bootstrap Icons
- Google Fonts (Playfair Display, Lato)
